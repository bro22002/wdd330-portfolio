  • Variables

    in programming are used to refer to values stored in the computer memory. In the JavaScript ES6 version, there are two ways to declare variables. Variables must be declared before they can be used in our code. To declare a variable, you can use const and the let keywords. The difference between the const and the let is, the const keyword is used when the value of the variable is not reassigned or changed while the let keyword is used when the value of the variable is reassigned later in the program. Below are examples of declaring variables using let and const.
    const name = “Brown”
    let number = 0

  • Scope

    is a concept in programming that refers to where a constant or variable is accessible in the program. Scopes can be either global or local.

    • Variables declared outside a block are said to have a global scope. That is, it can be used everywhere in the program.
    • Variables declared inside a block using the let and const are said to have a local scope. That is, their values are limited inside their blocks and cannot be accessed from the outside.

  • Template Literals

    this is a special type of string that uses the backtick character to enclose a string. The advantage of using a template literal is its ability to use both types of quotation marks. Here is an example of using a template literal. `Hello World`

  • Arrays

    are built-in objects that store an ordered list of values. In JavaScript, an array can be created using the following snippet const myArray = []; JavaScript has powerful methods to be used by arrays.

    • Pop() removes the last item from an array. Ex: myArray.pop();
    • The shift() method removes the first item of an array. Ex: myArray.shift();
    • The push() method appends a new value to the end of the array. Ex: myArray.push();
    • The unshift() method appends a new value at the beginning of the array

    Function Declaration
    function greetings(){
    console.log('Hello World!');
    }
    Function Expression
    const goodbye = function(){
    console.log('Goodbye World!');
    };
    Arrow Function
    const hello = () => alert('Hello World!');
